This project is an Interactive Study Tool for Economics, inspired by NotebookLM, designed to help students understand concepts through AI-based explanations, audio responses, and curated video summaries.

The tool allows students to:
Ask questions from a specific Economics textbook chapter
Get teacher-style AI explanations
Listen to responses via text-to-speech
Learn visually through YouTube video summaries
